AppGrid

Or choose a grid size.


AppGrid is a small (1.98KB), pure CSS boilerplate intended for use in desktop-style HTML5 apps. It's a convenient way to make use of both the horizontal and vertical space of a page - in the familiar fashion of a CSS grid system.


Coupled with something like node-webkit it's a great first step to building desktop applications in HTML5. But nothing's stopping you from running it in-browser either, of course.


It's much easier to show than tell, so here are some samples:

  • Basic Example (demonstration)
  • AppFiddle (jsfiddle layout in just a few lines, demonstrates nesting)
  • AppShop (image editor layout, ugly but you get the jist)


The code looks a little something like this: 
<div class="row size12 offset0">
	<div class="column size6 offset0">
		column 1
	</div>
	<div class="column size6 offset6">
		column 2
	</div>
</div>
